■ アプリ概要
施策一覧（Excel）をもとに、転記用フォーマットのExcelを自動生成するツール。
媒体ごとにシートを分割し、日別×施策別で稼働状況を可視化した転記用データを生成する。

■ 入力ファイル（施策一覧）
ユーザーがアップロードするExcelファイル。
想定構造

A列：媒体名 または 開始日
B列：終了日
C列：施策名
D列以降：使用しない（無視）

データの構造

A列に文字列（例：媒体A）が来た行 → 媒体の開始（ヘッダ）
その下の行で

A列：開始日
B列：終了日
C列：施策名
→ が繰り返される




■ 出力ファイル（転記用Excel）
テンプレート（転記用FMT.xlsx）をベースに生成。
出力仕様

媒体ごとにシート分割
シート名 = 媒体名
A列：日付（yyyy/mm/dd）
G列以降：施策ごとの列

データ内容

対象期間内を日単位で展開
各日付 × 各施策について

期間内 → 1
期間外 → 0




■ 主な処理内容

施策一覧を読み込み
A列の文字列から媒体を判定
媒体ごとに施策データ（開始日・終了日・施策名）を取得
媒体ごとに

施策名をユニーク抽出
日付を最小〜最大で展開
日別×施策でフラグ付与（1/0）


転記用FMTをコピーしてシート生成
Excelとして出力


■ UI機能

施策一覧ファイルのアップロード
対象シート選択
実行ボタン
ダウンロードボタン
完了時にエフェクト（🎈）


■ 出力ファイル名
転記用_[選択したシート名].xlsx

例：
転記用_施策一覧（2026年5月）.xlsx


■ 注意点

A列の媒体名と日付が混在する構造に依存している
日付はExcel形式または「yyyy/mm/dd」形式推奨
媒体名は31文字以内（Excel制限）


■ 想定用途

広告施策の転記・可視化
日別配信状況の整理
社内レポートや進捗管理用データ作成


■ 備考

入力データの構造が変わるとパースできなくなる可能性あり
必要に応じて列位置や判定ロジックの調整が必要
